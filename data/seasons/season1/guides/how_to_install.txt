Инструкция по установке модов, конфигов и прочего.
1. Перейдите по [$вечной ссылке](https://drive.google.com/drive/folders/1arr_tTa6CjzePjv75bWDG7Q_Lo2EJTZi?usp=sharing) на гугл-диск;
2. Перейдя по ссылке, вы увидите несколько папок и зип-архив. Вас интересуют файл *full_pc.zip*. Скачайте его, нажав по нему ПКМ;
3. Откройте архив, в нём будут папки **mods** и **config**. Зайдите в папку с установленной игрой. Там - удалите одноимённые папки, т.е. **mods** и **config**, после чего на их место поставьте новые, из архива.

Важное: эти папки могут отсутствовать в папке с установленной игрой, если вы ни разу её не запускали. Это нормально: следуйте инструкции как если бы вы уже удалили эти папки.
-----
Примечание: я выкладываю сборки в следующих видах. Всё идёт в виде архива для удобства скачивания, т.к. иногда гугл-диск ругается при скачивании большого кол-ва файлов напрямую из папок.
**full_pc** - стандартная сборка, моды + конфиги.
В папке **for_debug**:
**config** - отдельно конфиги к сборке.
~~**nude** - "Голая" сборка, исключительно необходимые моды без клиентских.~~
~~**potato** - "Голая" сборка + моды на оптимизацию.~~
~~**toster** - "Голая" сборка + моды на оптимизацию + моды на информацию (Вроде REI, EMI, Jade, пр.)~~
**laptop** - "Голая" сборка + моды на оптимизацию + моды на информацию + моды на удобство игры (Вроде mouse wheelie, mouse tweaks, пр.)
Сборка **full_pc**, соответственно, это "голая" сборка + моды на оптимизацию + моды на информацию + моды на удобство игры + моды на визуал.
Ответственности за сборки в папке **for_debug** не несу, они созданы исключительно для помощи людям, которые пытаются разобраться с техническими проблемами, с моей помощью или без.

-----

Поскольку моддинг Майнкрафта как искусство, то определить точные минимальные технические требования к сборке - сложно. Впрочем, ниже представлены те минимальные требования, которые были выведены на основе опыта игры игроков.
**Учитывайте, что для игры при подобных параметрах необходимо выключить абсолютно все визуальные эффекты, которые вызывают лаги - что можно узнать только на опыте - пошаманив с аргументами запуска Java и установив Laptop версию игры**, но это всё позволяет гарантированно играть в 50-60 ФПС даже при машине-картошке. Данная глава и в целом дополнение служит просто для ознакомления ||(И моего личного рекорда оптимизации сборки)||.

### Минимальные требования
- Винда, 64-битная версия (32-битная не подходит из-за нужды в Java 21+)
- ОЗУ: 8 гигов всего, 2.3 выделено под систему, +- 3 уходит под программы. Для Майнкрафта выделено 3 гига оперативной памяти
- Видео-карта: встроенная AMD Radeon
- Процессор: AMD Ryzen 5 7520u with radeon graphics (4 ядра, 8 потоков, 2.8 Гигагерц до 4.3)
